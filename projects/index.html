Projects

Posted at — Jan 1, 0001

I don’t have any major projects going on at the moment, but here’s what I’ve worked on in the past:

  • Way Cooler: A Wayland clone of AwesomeWM for Wayland.
  • todo: Personal task manager shaped to the way I deal with and think of tasks
  • wlroots-rs: A 100% safe Rust wrapper for wlroots, a library to build Wayland compositors.
  • rust-wlc: A Rust wrapper for wlc. It is suggested to use wlc.rs or wlroots-rs instead.
  • autowikiabot-py: A Reddit bot written in Python that gave summaries of links that were posted to the http://wikia.com domain.
  • Wikia: A python wrapper for the Wikia API. Made for use in autowikiabot.
